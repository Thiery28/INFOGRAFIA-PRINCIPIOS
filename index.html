1. Introducción

En el ámbito de la programación, el dominio de estructuras condicionales y operaciones básicas con variables constituye la base fundamental para el desarrollo de algoritmos más complejos. El presente informe tiene como objetivo documentar el desarrollo de seis ejercicios prácticos implementados en cinco lenguajes de programación diferentes: C++, Java, Python, C# y JavaScript/TypeScript. La elección de estos lenguajes responde a su relevancia en el panorama actual de la programación, abarcando desde lenguajes de propósito general hasta aquellos especializados en desarrollo web y de sistemas. Cada ejercicio aborda conceptos fundamentales como la toma de decisiones mediante estructuras condicionales (if-else), operaciones aritméticas, manejo de entrada/salida por consola, y procesamiento de datos numéricos. Este trabajo ha sido desarrollado de manera colaborativa por un equipo de programadores, donde cada miembro se especializó en la implementación de los ejercicios en un lenguaje específico, permitiendo así un análisis comparativo enriquecido por las particularidades sintácticas y paradigmáticas de cada tecnología.

2. Objetivos

2.1. Objetivo General

Desarrollar e implementar seis algoritmos fundamentales de programación en cinco lenguajes distintos (C++, Java, Python, C# y JavaScript/TypeScript), demostrando el dominio de estructuras condicionales, operaciones aritméticas y manejo de entrada/salida por consola.

2.2 Objetivos Específicos

  • 1. Implementar un programa de verificación de mayoría de edad utilizando condicionales simples.
  • 2. Desarrollar un algoritmo de intercambio de valores entre variables siguiendo reglas específicas.
  • 3. Crear un sistema de evaluación de notas con mensajes personalizados según el resultado.
  • 4. Programar un identificador de números pares e impares mediante el operador módulo.
  • 5. Diseñar un analizador completo de números enteros que evalúe múltiples características simultáneamente.
  • 6. Implementar la operación de suma de números complejos, trabajando con partes reales e imaginarias.
  • 7. Comparar las implementaciones entre los diferentes lenguajes para identificar similitudes y diferencias sintácticas.
  • 8. Documentar adecuadamente cada solución siguiendo estándares de calidad y formato APA.

Ejercicio 1: Verificación de Mayoría de Edad
C++
#include <iostream> using namespace std; int main() { int edad; cout << "Ingrese su edad: "; cin >> edad; if (edad >= 18) { cout << "Eres mayor de edad." << endl; } else { cout << "Eres menor de edad." << endl; } return 0; }
Java
import java.util.Scanner; public class MayoríaEdad { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.print("Ingrese su edad: "); int edad = sc.nextInt(); if (edad >= 18) { System.out.println("Eres mayor de edad."); } else { System.out.println("Eres menor de edad."); } sc.close(); } }
Python
edad = int(input("Ingrese su edad: ")) if edad >= 18: print("Eres mayor de edad.") else: print("Eres menor de edad.")
C#
using System; class MayoríaEdad { static void Main() { Console.Write("Ingrese su edad: "); int edad = int.Parse(Console.ReadLine()); if (edad >= 18) { Console.WriteLine("Eres mayor de edad."); } else { Console.WriteLine("Eres menor de edad."); } } }
JavaScript/TypeScript
// JavaScript (Node.js) const readline = require('readline').createInterface({ input: process.stdin, output: process.stdout }); readline.question('Ingrese su edad: ', edad => { if (parseInt(edad) >= 18) { console.log('Eres mayor de edad.'); } else { console.log('Eres menor de edad.'); } readline.close(); });

C++

Java
public class Intercambio { public static void main(String[] args) { int a = 5, b = 10, temp; System.out.println("Antes: a=" + a + ", b=" + b); temp = a; a = b; b = temp; System.out.println("Después: a=" + a + ", b=" + b); } }
Python
a, b = 5, 10 print(f"Antes: a={a}, b={b}") # Intercambio sin variable temporal (Python style) a, b = b, a print(f"Después: a={a}, b={b}")
C#
using System; class Intercambio { static void Main() { int a = 5, b = 10, temp; Console.WriteLine($"Antes: a={a}, b={b}"); temp = a; a = b; b = temp; Console.WriteLine($"Después: a={a}, b={b}"); } }

C++

Java

Python
nota = float(input("Ingrese la nota (0-20): ")) if nota < 0 or nota > 20: print("Nota inválida") elif nota >= 11: print("Aprobado") else: print("Reprobado")
C#
using System; class Notas { static void Main() { Console.Write("Ingrese la nota (0-20): "); double nota = double.Parse(Console.ReadLine()); if (nota < 0 || nota > 20) { Console.WriteLine("Nota inválida"); } else if (nota >= 11) { Console.WriteLine("Aprobado"); } else { Console.WriteLine("Reprobado"); } } }

const readline = require('readline').createInterface({ input: process.stdin, output: process.stdout }); readline.question('Ingrese la nota (0-20): ', nota => { const n = parseFloat(nota); if (n < 0 || n > 20) { console.log('Nota inválida'); } else if (n >= 11) { console.log('Aprobado'); } else { console.log('Reprobado'); } readline.close(); });

C++
#include <iostream> using namespace std; struct Complejo { double real; double imag; }; Complejo sumar(Complejo a, Complejo b) { Complejo resultado; resultado.real = a.real + b.real; resultado.imag = a.imag + b.imag; return resultado; } int main() { Complejo c1, c2, suma; cout << "Primer número complejo:" << endl; cout << "Parte real: "; cin >> c1.real; cout << "Parte imaginaria: "; cin >> c1.imag; cout << "Segundo número complejo:" << endl; cout << "Parte real: "; cin >> c2.real; cout << "Parte imaginaria: "; cin >> c2.imag; suma = sumar(c1, c2); cout << "La suma es: " << suma.real << " + " << suma.imag << "i" << endl; return 0; }
Java
import java.util.Scanner; class Complejo { double real; double imag; Complejo(double real, double imag) { this.real = real; this.imag = imag; } static Complejo sumar(Complejo a, Complejo b) { return new Complejo(a.real + b.real, a.imag + b.imag); } } public class SumaComplejos { public static void main(String[] args) { Scanner sc = new Scanner(System.in); System.out.println("Primer número complejo:"); System.out.print("Parte real: "); double r1 = sc.nextDouble(); System.out.print("Parte imaginaria: "); double i1 = sc.nextDouble(); System.out.println("Segundo número complejo:"); System.out.print("Parte real: "); double r2 = sc.nextDouble(); System.out.print("Parte imaginaria: "); double i2 = sc.nextDouble(); Complejo c1 = new Complejo(r1, i1); Complejo c2 = new Complejo(r2, i2); Complejo suma = Complejo.sumar(c1, c2); System.out.println("La suma es: " + suma.real + " + " + suma.imag + "i"); sc.close(); } }

3. Conclusiones

1. C++ demostró ser un lenguaje robusto y de alto rendimiento, ideal para aplicaciones que requieren control fino sobre la memoria y el procesamiento. Su sintaxis, aunque compleja, permite expresar operaciones de manera muy explícita y eficiente. La implementación orientada a objetos para números complejos mostró la madurez de sus capacidades de abstracción.

2. La implementación de los mismos algoritmos en diferentes lenguajes evidencia que, aunque la sintaxis varía significativamente, la lógica fundamental de la programación estructurada permanece constante.

3. Los lenguajes de tipado estático (C++, Java, C#) ofrecen mayor seguridad en tiempo de compilación, mientras que los de tipado dinámico (Python, JavaScript) proporcionan mayor flexibilidad y rapidez en el desarrollo.

4. La comprensión de estructuras condicionales y operaciones básicas es fundamental independientemente del lenguaje de programación elegido.
